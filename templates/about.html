About

Find emerging markets for athletes/sports using twitter data.

For 180 countries Tweete-lete recommends the top 5 athletes in that particular country.

Data

  1. Twitter API tweepy used to get 14 GB of data.
  2. The athlete names, their sport, country and gender, were scraped from ESPN.

Sentiment Mining

The tweet text is given a polarity from -1 to 1, with anything above 0 positive, below 0 negative, and everything else is neutral.

  1. A Naive Bayes [1] analysis is used for classification on an n-gram model[2] of the text.
  2. TextBlob library was used. The classification is reported to have an accuracy of 0.8.
  3. Some examples of the classification:

    1. Positive: 11 years ago today, Kobe Bryant absolutely destroyed Steve Nash.
    2. Negative: They might've been worse than Iverson 2001 Sixers team...
    3. Neutral: Remember when Serena williams beat sharapova and a Nigerian paper's headline the next day was "Sharap it's ova"

Recommender Engine

  1. Collaborative Filter is used predict the most-loved players.
  2. Countries form the customer base
  3. Athletes form the products
  4. The sentiment score is used as the rating
